# Setup

The code can be run by setting a venv or directly using the python3
version installed in the csc servers.

## Method 1

1. setup the virtual environment (venv)
```
    #choose the appropriate python alias for your system 
    python -m venv ./venv
    python3 -m venv ./venv
``` 
2. activate the venv ```source ./venv/bin/activate```
3. run ```python WebTester.py <uri>

## Method 2

```python3 WebTester.py <uri>```
